# User Preference

The `@ldn-viz/ui` package exposes wrappers around MediaQuery from [svelte/reactivity](https://svelte.dev/docs/svelte/svelte-reactivity#MediaQuery).

- `prefersDarkMode()` is a convenience wrapper around `mediaQuery`: it returns provides a 'current' property that reflects whether the user has set [prefers-color-scheme](https://developer.mozilla.org/en-US/docs/Web/CSS/@media/prefers-color-scheme) to `dark`

- `prefersReducedMotion()` is a convenience wrapper around `mediaQuery`: it returns a provides a 'current' property that reflects whether the user has enabled the [prefers-reduced-motion](https://developer.mozilla.org/en-US/docs/Web/CSS/@media/prefers-reduced-motion) setting

## Example use

```js
import { prefersDarkMode } from '@ldn-viz/ui';

$inspect('Dark mode is', prefersDarkMode.current ? 'enabled' : 'disabled');
```
