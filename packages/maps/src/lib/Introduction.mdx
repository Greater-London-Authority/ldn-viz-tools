# Welcome to Storybook for ldn-viz-tools maps

The`maps` package currently provides two things:

- a set of useful constants (such as a bounding box for Greater London)
- a set of maplibre style files for commonly-used basemaps

### Geocoding

When geocoding using the [mapbox-gl-geocoder](https://github.com/mapbox/mapbox-gl-geocoder), you can specify a bounding box to restrict searches to.
This prevents users form being annoyed or distracted by irrelevant suggestions.
For example:

```js
import { GREATER_LONDON_BOUNDS_FLAT } from '@ldn-viz/maps';

const geocoder = new MapboxGeocoder({
	accessToken: mapboxgl_accessToken,
	mapboxgl: maplibregl,
	bbox: GREATER_LONDON_BOUNDS_FLAT
});

map.addControl(geocoder);
```
